How to insert API key

Remember: Before inserting the API key make sure you have installed Vubla’ Magento extension (download it here).

  1. Find and copy your API key from step 3 in Vubla’ installation process.

  2. Login to your Magento admin panel.

  1. Go to System -> Web Services -> Users.

  1. Press on the “Vubla” user:

  2. Paste in your API key in fields “New API Key” and “API Key Confirmation”. Press “Save user”.

  1. Completed! Now, wait for Vubla to crawl your webshop that can take up till 5 hours depending on the number of products your webshop has. Vubla will send you an e-mail when it finishes crawl and you can continue the installation process.
